# 🧮 Java 계산기 과제
<br>

### STEP 1. 클래스 없이 기본적인 연산을 수행할 수 있는 계산기 만들기
<ul>
    <li><b>양의 정수(0 포함) 입력받기</b>
        <br>Scanner를 활용해 두 개의 양의 정수와 사칙연산 기호(+, -, *, /)를 입력받아 변수에 저장함.
    </li>
    <li><b>입력받은 양의 정수 2개와 사칙연산 기호를 사용하여 연산 진행 후 결과값 출력</b>
        <br>switch문으로 입력받은 연산 기호에 따라 구분하여 사칙연산을 진행하고 결과값을 출력
    </li>
    <li><b>예외 상황 처리</b>
        <br>나눗셈 연산 시 분모에 0이 입력될 경우 에러 메시지를 출력
    </li>
    <li><b>“exit” 문자열을 입력하기 전까지 무한으로 계산을 진행</b>
        <br>while 반복문을 사용하여 사용자가 "exit"을 입력하기 전까지 계산을 계속해서 수행할 수 있도록 구현
    </li>
</ul>

<br>

### STEP 2. 클래스를 적용해 기본적인 연산을 수행할 수 있는 계산기 만들기       
<ul>
    <li><b>결과값 반환 메서드 구현 & 연산 결과를 저장하는 컬렉션 타입 필드를 가진 Calculator 클래스를 생성</b>
        <br>사칙연산을 수행한 후, 결과값을 반환하는 메서드 구현, Calculator 클래스를 생성
    </li>
    <li><b>main 메서드에 Calculator 클래스가 활용</b>
        <br>연산 수행과 결과를 Calculater 클래스에서 함
    </li>
    <li><b>Calculator 클래스의 연산 결과를 저장하고 있는 컬렉션 필드에 직접 접근하지 못하도록 수정 (캡슐화)</b>
        <br>getter, setter 메서드 구현, App 클래스의 main 메서드에서 위에서 구현한 메서드를 활용
    </li>
    <li><b>가장 먼저 저장된 데이터를 삭제하는 기능을 가진 메서드를 구현</b>
        <br>컬렉션에서 가장 먼저 저장된 값을 삭제
    </li>
</ul>
<br>
      
### STEP 3. Enum, 제네릭, 람다 & 스트림을 이해한 계산기 만들기
<ul>
    <li><b>Enum 타입을 활용</b>
        <br>Enum 타입을 활용하여 연산자 타입에 대한 정보를 관리한다
    </li>
    <li><b>double 타입의 값을 전달 받아도 연산이 수행하도록 만들기</b>
        <br>제네릭을 이용해 double 타입의 값을 전달받아도 계산을 할 수 있게 수정
    </li>
    <li><b>Scanner로 입력받은 값보다 큰 결과값 들을 출력</b>
        <br>Lambda & Stream을 활용하여 구현
    </li>
</ul>
